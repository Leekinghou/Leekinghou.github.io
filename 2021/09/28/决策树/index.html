决策树

决策树是根据规则，对实例进行分类的树形结构。分叉应该是互斥的、完备的。

决策树模型

if-then

构建步骤

  1. 构建根节点
  2. 选择最优特征，以此分割训练数据集
  3. 若子集被基本正确分类，构建叶节点，否则，继续选择新的最优特征
  4. 重复2、3，知道所有的数据子集被正确分类

最优：能将训练数据集大部分样本正确分类就行。

因为模型选择不仅要拟合，还要考虑对未知数据的泛化。

如何选择：剪枝

概率模型

  1. 朴素贝叶斯
  2. 决策树

（非概率模型：感知机、k近邻）

条件概率分布

决策树：给定特征条件下类的P(Y|X)

条件概率分布：特征空间的一个划分（Partition）

划分：单元（Cell）或区域（Region）互不相交

假设只有两类：正类和负类，即Y的取值为+1和-1，（a）中小矩形中的数字表示单元的类。（b）表示特征空间划分确定时，特征（单元）给定条件下类的条件概率分布。（b）中条件概率对应（a）的划分。当某个单元c的条件概率满足P(Y = +1|X = c) > 0.5时，则认为这个单元属于正类。（c）为对应（b）中条件概率分布的决策树。

对于图（c）：

  1. 一条路径对应于划分中的一个单元
  2. 决策树的条件概率分布由各个单元给定条件下类的条件概率分布组成

决策树学习

  1. 本质是从训练集中归纳出一组分类规则，与训练集不相矛盾
  2. 假设空间：由无穷多个条件概率模型组成
  3. 一个好的决策树：能完美拟合已知数据又能泛化未知数据

实现思路

  1. 策略是最小化损失函数
  2. 特征选择：递归选择最优特征
  3. 生成：对应特征空间的划分，直到所有训练子集被基本正确分类
  4. 剪枝：避免过拟合，具有更好的泛化能力

熵

信息增益表示得知特征X的信息而使得类Y的信息的不确定性减少的程度。

  • 因此随机变量的取值等概率分布的时候，相应的熵最大

  • 当熵和条件熵中的概率由数据估计得到时，则为经验熵和经验条件熵。

信息增益

  • 信息增益：得知特征X而使类Y的信息的不确定性减少的程度

    • A是训练集中的特征
    • 因此哪个特征A所带来的信息增益最大，就选哪个作为最优特征。（作用）
